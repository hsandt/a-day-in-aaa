# A Day in AAA

A Visual Novel made with Ren'Py about a junior game programmer working in AAA company

## Progress

* Project setup
* A few scene drafts

## Tools

* Ren'Py 8.0.3
* Sublime Text
* [Game Title Generator](https://letsmakeagame.net/game-title-generator/)
